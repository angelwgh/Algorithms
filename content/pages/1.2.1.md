## 插入排序 

我们的第一个算法（插入排序）：

**输入：** n个数的一个序列$(a_1,a_2,a_3,···,a_n)$。

**输出：** 输入序列的一个排列$(a_1^{\prime},a_2^{\prime},···,a_n^{\prime})$，满足$a_1^{\prime}{\leq}a_2^{\prime}{\leq}···{\leq}a_n^{\prime}$。

```js
function paixu(A){
    for (var i=1 ; i <= A.length-1 ; i++){
        var key = A[i];
        var j = i-1;
        while(j>=0 && A[j]>key){
            A[j+1] = A[j]
            j=j-1;
        }
        A[j+1]=key;
    }
    return A;
}
```
